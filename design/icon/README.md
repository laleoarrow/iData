# iData Icon v1

## Direction

Chosen direction: **line-tech data glyph**.

- `i` is expressed as a glowing dot + stem
- `data` is expressed as a rounded grid/table outline
- dark glass background keeps it close to current macOS visual language
- thin luminous strokes keep the icon minimal and technical instead of cartoon-like

## Files

- `idata-icon-v1.svg`: editable master vector
- `idata-icon-v1.png`: 1024 preview rendered from the SVG
- `AppIcon.appiconset/*`: app icon payload for Xcode
