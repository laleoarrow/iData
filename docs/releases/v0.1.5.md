# iData v0.1.5

Released: 2026-03-08

## Highlights

- Fixed a tutorial-session rendering regression where the first VisiData toolbar/menu row could disappear and cause visual misalignment.
- Improved embedded terminal display readiness handling to avoid repeated transcript replay during rebind/update cycles.
- Refined collapsed-sidebar interactions:
  - destructive actions now require `Command` hover intent more consistently
  - click behavior in narrow history mode is more reliable
- Updated status/input area layout to a single unified status panel style with clearer English-input indication.
- Slowed the full-window shimmer sweep to reduce distraction and improve perceived smoothness.

## Notes

- Build remains unsigned/not notarized for local distribution.
