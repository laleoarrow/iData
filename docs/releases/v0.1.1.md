🍎 `iData v0.1.1` brings the first polished in-window `VisiData` experience.

- adds a glass-style welcome screen with version, format examples, and common VisiData shortcuts
- opens most regular files directly, including odd bioinformatics suffixes such as `.ma`
- streams `.gz` / `.bgz` / `.bgzf` files without extracting them to disk
- adds Sparkle-based in-app update checks backed by GitHub Releases and `appcast.xml`
- packages a drag-to-Applications `.dmg` installer alongside the `.zip` bundle
