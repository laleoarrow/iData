🍎 `iData v0.1.4` completes the current polish and packaging cycle.

- tightens session shutdown so terminating `iData` also stops descendant `VisiData` processes reliably
- adds a regression test for descendant-process cleanup
- closes the app when the last window closes, matching standard macOS utility behavior
- upgrades the landing page into a fuller product site with clearer release and download presentation
- adds a `.pkg` installer that installs `iData.app` and drops a post-install `pipx + visidata + openpyxl` setup guide
- keeps the native macOS shell, embedded `VisiData` workflow, Sparkle appcast path, and recent-file UX from `v0.1.3`
