🍎🎉 `iData v0.1.0`

First public release of `iData`, a native macOS shell for opening large tables with `VisiData`.

## Included in v0.1.0

- Native macOS window with recent files and drag-and-drop
- Embedded terminal-based `VisiData` session inside the app
- Supported formats:
  - `csv`
  - `tsv`
  - `txt`
  - `json`
  - `jsonl`
  - `xlsx`
  - `csv.gz`
  - `tsv.gz`
  - `txt.gz`
- Recent-file management, including single-item remove and clear-all
- Configurable `vd` executable path in Preferences
- Improved startup/session refresh behavior for embedded terminal rendering

## Dependency

`VisiData` is required and is not bundled in this release.

Recommended install:

```bash
brew install visidata
```

Or set a custom `vd` path in Preferences after launching `iData`.

## Install

1. Download `iData-v0.1.0-macos-universal.zip`
2. Unzip it
3. Move `iData.app` into `/Applications`
4. Launch `iData`

If macOS blocks the first launch, open it from `System Settings > Privacy & Security`.
